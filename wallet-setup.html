Flare Network · MetaMask · Bifrost · Ledger (Hardware)

Eligibility note: Energon participation is enforced programmatically. Only wallets holding exactly one Energon Cube are eligible. Wallets holding 0 or more than 1 cube are excluded.

Option A — MetaMask

  1. Install MetaMask (browser extension or mobile) and create/import your wallet.
  2. Add Flare Network manually in MetaMask:
    Network Name
    Flare Mainnet
    New RPC URL
    https://flare-api.flare.network/ext/C/rpc
    Chain ID
    14
    Currency Symbol
    FLR
    Block Explorer
    https://flare-explorer.flare.network
  3. Switch MetaMask to Flare Mainnet.
  4. Add EON token so it shows in your wallet:
    0x9458Cbb2e7DafFE6b3cf4d6F2AC75f2d2e0F7d79
  5. After purchasing an Energon Cube: MetaMask may not display NFTs automatically. Verify ownership on explorer:
    View CUBE on Explorer
  6. Connect to Energon dApps and approve the wallet connection prompt.

Option B — Bifrost Wallet

  1. Install Bifrost Wallet on mobile and create/import your wallet.
  2. Select Flare network inside Bifrost (not Songbird or other networks).
  3. Add EON token: Find “Manage tokens” / “Add token” and paste:
    0x9458Cbb2e7DafFE6b3cf4d6F2AC75f2d2e0F7d79
  4. After purchasing an Energon Cube: Look under NFTs/Collectibles. If it doesn’t appear immediately, confirm via explorer:
    Verify CUBE on Explorer
  5. Connect using WalletConnect: On an Energon site, click Connect → WalletConnect → scan the QR code with Bifrost.

Option C — Ledger / Hardware Wallet (Recommended for Security)

A hardware wallet protects your private keys offline. The most common setup on EVM chains like Flare is: Ledger + MetaMask. You manage accounts in Ledger, and MetaMask acts as the interface.

  1. Set up Ledger using Ledger Live, create your device PIN, and store your recovery phrase safely (offline).
  2. Install the Ethereum app on your Ledger using Ledger Live.
    Flare is EVM-compatible, so Ledger uses the Ethereum app for signing.
  3. Connect Ledger to MetaMask: In MetaMask → Accounts → “Add account” → Hardware wallet → choose Ledger → connect and select an address.
  4. Add Flare Network in MetaMask (same fields as Option A) and switch to Flare Mainnet.
  5. Add EON token in MetaMask (same token address):
    0x9458Cbb2e7DafFE6b3cf4d6F2AC75f2d2e0F7d79
  6. Purchase / hold your Energon Cube on the Ledger account you connected. Verify ownership on explorer if the NFT does not show in-wallet:
    Verify CUBE on Explorer
  7. When signing a transaction, always verify the address + details on the Ledger screen before approving.
Hardware wallet safety: Never type your Ledger recovery phrase into any website or app. Energon will never ask for it. Only enter it on your Ledger device during recovery.

Troubleshooting

  • Balances look empty: switch wallet network to Flare Mainnet.
  • EON not showing: re-add the token using the EON contract address.
  • NFT not visible: check the explorer inventory/holder view to confirm ownership.
  • Ledger won’t connect: unlock device, open Ethereum app, enable “Blind signing” if prompted by Ledger/MetaMask.
