Option A ‚Äî MetaMask

  1. Install MetaMask and create or import your wallet.
  2. Add the Flare Mainnet network using the network details above.
  3. Switch MetaMask to Flare Mainnet.
  4. Fund your wallet with FLR (for gas + mint): Tap Receive in MetaMask to copy your address, then send/withdraw FLR (Flare Mainnet) to that address.
  5. Add EON in MetaMask (mobile): Go to Tokens ‚Üí tap + (Import tokens) ‚Üí select Custom token ‚Üí paste the EON contract address:
    0x9458Cbb2e7DafFE6b3cf4d6F2AC75f2d2e0F7d79
    EON should auto-populate ‚Üí tap Import. (EON will now appear in your wallet.)
  6. Tap and copy the Energon dApp address:
    https://energon-dapp.vercel.app
  7. Open the dApp Browser inside the MetaMask app.
  8. Paste the Energon dApp address into the address bar of the dApp Browser and load the page.
  9. Tap Connect Wallet on the Energon dApp.
  10. Approve the wallet connection request in MetaMask.
  11. Confirm the network is Flare Mainnet (MetaMask will prompt you to switch if required).
  12. Follow the on-screen steps to mint or purchase one Energon Cube.
  13. Review the transaction details carefully and approve the transaction in MetaMask.
  14. After confirmation, if the Energon Cube does not appear immediately, verify ownership on the Flare Explorer:
    Verify CUBE on Explorer
If the mint site opens in Safari/Chrome instead of MetaMask, copy the URL and open it inside the MetaMask dApp Browser.

Option B ‚Äî Bifrost Wallet

  1. Install Bifrost Wallet and create or import your wallet.
  2. Ensure the active network is set to Flare Mainnet (or Flare).
  3. Fund your wallet with FLR (for gas + mint): Tap Receive to copy your Flare address, then send/withdraw FLR (Flare Mainnet) to that address.
  4. Add EON in Bifrost (manual import):
    Tap the blue up arrow ‚Üí tap Receive ‚Üí search Energon ‚Üí tap Custom token ‚Üí make sure Flare Mainnet is selected (Flare at the top) ‚Üí paste the EON contract address:
    0x9458Cbb2e7DafFE6b3cf4d6F2AC75f2d2e0F7d79
    You should see EON populate ‚Üí confirm/add. (EON will now show in your token list.)
  5. Tap and copy the Energon dApp address:
    https://energon-dapp.vercel.app
  6. Open the dApp Browser inside the Bifrost app.
  7. Paste the Energon dApp address into the address bar of the dApp Browser and load the page.
  8. Tap Connect Wallet on the Energon dApp.
  9. Approve the wallet connection inside Bifrost (this may appear as an in-app approval or WalletConnect confirmation).
  10. Confirm the network is Flare Mainnet.
  11. Follow the on-screen steps to mint or purchase one Energon Cube.
  12. Approve the transaction in Bifrost.
  13. If the Energon Cube does not appear immediately, verify ownership on the Flare Explorer:
    Verify CUBE on Explorer

Option C ‚Äî Ledger (Hardware Wallet)

Ledger provides maximum security by keeping private keys offline. Use Ledger together with MetaMask for Flare compatibility.

  1. Initialize Ledger and store your recovery phrase securely offline.
  2. Install the Ethereum app on Ledger.
  3. Connect Ledger to MetaMask as a hardware wallet.
  4. Add Flare Network in MetaMask (using the network details above).
  5. Fund with FLR: Send/withdraw FLR (Flare Mainnet) to your Ledger address (view/copy it in MetaMask ‚Üí Receive).
  6. Add EON in MetaMask: Go to Tokens ‚Üí + (Import) ‚Üí Custom token ‚Üí paste:
    0x9458Cbb2e7DafFE6b3cf4d6F2AC75f2d2e0F7d79
    then Import.
  7. Open the Mint Site in MetaMask (dApp Browser) and always verify transaction details on the Ledger device screen before approving.
Never share your Ledger recovery phrase. Energon will never request it.
