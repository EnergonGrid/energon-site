Deterministic Protocol
Defined by rules. Observed by Guardians.
Energon is a live on-chain system operating on Flare mainnet. The protocol advances through public contract state, not hidden automation. The interface only reflects what the contracts expose.
New Here?

The Protocol
The system follows conditions, not commands.
Energon does not rely on administrators, external operators, or hidden automation. Every meaningful transition occurs only when the required on-chain conditions are satisfied.
Verification
Everything can be observed.
The protocol state exists publicly on Flare. Transactions, supply, Guardian states, and system activity can be independently verified. The interface does not control the protocol; it simply displays its current state.
The Guardian Rule
One wallet. One cube.
The system does not interpret intention. It reads state. A wallet is coherent only when it holds exactly one Energon Cube.
